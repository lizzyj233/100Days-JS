JS Golf Project Beginnings

May 15-17, 2021

I began a JS project to calculate the slope/rating for tees at a golf course which are not already rated for female golfers. This is something I was inspired to make as a female golfer who likes to play the longer tees for a challenge. Often times, these tees are not rated for female golfers so I have to manually calculate the difficulty... so I figured... why not make my life easier and turn this into a project?!? Please see my repo at GitHub to see the work i'm doing on this project. I'll probably hang out here for a while because this is something I personally need as golf season is upon us!

Ideally, I'll create the JS version (yes, it needs a lot of work - those elifs are super inefficient... gotta think through a loop) and a swift version which I can load up to my phone. I'm really excited about this project and think it could be extremely useful to many female golfers who don't prefer to play the forward tees :) -- @LPGA you wanna sponsor my project :D
